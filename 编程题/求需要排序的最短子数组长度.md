# 求需要排序的最短子数组长度


## 【题目】

对于一个无序数组A，请设计一个算法，求出需要排序的最短子数组的长度。
给定一个整数数组A及它的大小n，请返回最短子数组的长度。
例如数组：[1,5,3,4,2,6,7]，则返回4

## 【代码实现】


```java
public class SortTest5 {


    public static void main(String[] args) {
        int[] arr = {1,2,10,1,8,9};
        int length = find(arr);
        System.out.println(length);
    }

    private static int find(int[] A) {
        int n = A.length;
        int right = 0;
        int left = A.length-1;
        int max = A[0];
        for (int i =0;i<n;i++){
            if(A[i]>=max){
                max = A[i];
            }else {
                right = i;
            }
        }
        if (right==0){
            return right;
        }

        int min = A[n-1];
        for (int j=n-1;j>-1;j--){
            if(A[j]<=min){
                min = A[j];
            }else {
                left = j;
            }
        }


        return right-left+1;
    }
```
