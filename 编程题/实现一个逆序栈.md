# 实现一个逆序栈


## 【题目】

实现一个栈的逆序，但是只能用递归函数和这个栈的本身操作来实现，而不能自己申请另外的数据结构。



## 【代码实现】
```java

import java.util.Stack;

public class StackTest1 {



    //利用递归将栈底元素取出，其他元素原顺序压会栈中
    private static int get(Stack<Integer> stack){
        int result = stack.pop();
        if (stack.isEmpty()){
            return result;
        }else {
            int last = get(stack);
            stack.push(last);
            return last;
        }
    }


    //栈的逆序
    private static void reverse(Stack<Integer> stack){
        if(stack.isEmpty()){
            return;
        }
        int i = get(stack);
        reverse(stack);
        stack.push(i);
    }

}


```
