## 【代码实现】

```java
public class StringTest6 {
    public static void main(String[] args) {
        String str1 = "abc";
        String str2 = "de";
        System.out.println((str1+str2).compareTo(str2+str1)<0?(str1+str2):(str2+str1));
        
        
        String str3 = "b";
        String str4 = "ba";
        System.out.println((str3+str4).compareTo(str4+str3)<0?(str3+str4):(str4+str3));
    }
}
```

运行结果：
```
abcde
bab
```
